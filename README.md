# ABF-Convert
Batch convert Axon ABF files to CSV / TSV.

<b>Usage:</b>  _python abfconvert.py_

<b>Installation:</b> Just download abfconvert.py and place it wherever you want.
  
<b>Requirements:</b>
  - Python3
  - Numpy
  - Pandas
  - Neo =>0.4.1
  - tkinter

This script might work with python 2 but no promises.
